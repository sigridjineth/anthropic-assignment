# Case Studies Skill

## Purpose

Provide compelling customer success stories and reference examples that help prospects understand the practical value and real-world applications of our product.

## When to Use This Skill

Activate this skill when the conversation includes:
- "Do you have customers in [industry]?"
- "Who else uses this?"
- "Can you share success stories?"
- "What results have other customers seen?"
- "Any references in [sector/size]?"
- "Case studies"
- "Similar companies"
- "Proof points"

## Answer Guidelines

1. **Match customer context**: Reference cases similar to the prospect's industry, size, or use case
2. **Lead with outcomes**: Quantifiable results first, then story
3. **Be specific but appropriate**: Use approved metrics and anonymized details where needed
4. **Offer next steps**: Reference calls, detailed case studies, etc.

## Response Format

```
"I can share a relevant example. [Company type/industry] in [context] achieved [key metric].
They were facing [challenge similar to prospect's] and after implementing our solution,
they saw [specific improvement]. [Offer: Would you like to speak with them / see the full case study?]"
```

## Key Metrics to Highlight

- Processing time reduction (%)
- Accuracy improvement (%)
- Cost savings ($)
- Time to value (weeks/months)
- Team productivity gains

## Do NOT

- Share customer names without permission
- Invent or exaggerate metrics
- Promise identical results
- Disclose confidential implementation details

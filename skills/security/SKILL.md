# Security Skill

## When to Use
- Questions about compliance (SOC2, HIPAA, GDPR, etc.)
- Data encryption and protection
- Access control and authentication
- Data residency requirements
- On-premises deployment questions
- Security audit/assessment questions

## Response Guidelines
1. **Lead with certifications**: SOC2 Type II, ISO 27001
2. **Be specific about encryption**: TLS 1.3 in transit, AES-256 at rest
3. **Address data residency directly**: We support US, EU, APAC regions
4. **Be honest about limitations**: If we don't support something, say so clearly

## Key Security Features

### Compliance
- SOC 2 Type II certified (annual audit)
- ISO 27001 certified
- GDPR compliant (EU data processing)
- HIPAA eligible (Business Associate Agreement available)

### Encryption
- In transit: TLS 1.3
- At rest: AES-256
- Key management: AWS KMS with customer-managed keys available

### Access Control
- SSO integration (SAML 2.0, OIDC)
- Role-based access control (RBAC)
- Multi-factor authentication (MFA)
- API key rotation policies

### Data Residency
- US region (default)
- EU region (Frankfurt)
- APAC region (Singapore) - Enterprise only

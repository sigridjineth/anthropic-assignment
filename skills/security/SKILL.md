# Security Skill

## Purpose
Provide accurate information about security practices, compliance status, and data protection measures.

## When to Use
- Compliance questions (SOC2, HIPAA, GDPR, etc.)
- Data handling and privacy questions
- Encryption and security architecture
- Audit and access control questions
- Vendor security assessments

## Answer Guidelines

### Be Specific About
1. **Certification status**: Current vs in-progress
2. **Data handling**: What we collect, store, retain
3. **Encryption**: At rest, in transit, key management
4. **Access controls**: Who can access what

### Compliance Responses
- Always confirm current certification status
- Offer to provide compliance documentation
- Mention annual audit cycles

### Example Response Format
"We are SOC 2 Type II certified, with our most recent audit completed in November 2024. All data is encrypted at rest using AES-256 and in transit using TLS 1.3. I can share our security whitepaper and provide access to our trust portal for detailed documentation."

- Share internal security architecture details
- Discuss specific security incidents
- Make claims about certifications in progress as if complete

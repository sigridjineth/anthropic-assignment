# Roadmap Messaging Guidelines

## Approved Phrases

### For GA Features
- "This feature is fully available today"
- "You can start using this immediately"
- "This is production-ready and fully supported"

### For Beta Features
- "This is currently in beta with select customers"
- "We expect GA in [quarter], though timelines may shift"
- "Would you like to join the beta program?"
- "Beta features are fully functional but may have limited support"

### For Planned Features
- "This is on our roadmap for [quarter]"
- "We're actively working on this"
- "Timelines are subject to change based on priorities"
- "I can add you to the notification list for updates"

### For Not Planned Features
- "This isn't currently on our roadmap"
- "We're always evaluating customer needs - I'll share this feedback"
- "Have you considered [alternative approach]?"

## Phrases to Avoid
- "I promise this will be ready by..."
- "This is definitely coming in..."
- "Just between us, we're working on..."
- "Off the record..."
- Any specific dates without caveat

## Escalation Triggers
If customer pushes for commitment on:
- Specific release dates
- Features not on public roadmap
- Custom development timelines

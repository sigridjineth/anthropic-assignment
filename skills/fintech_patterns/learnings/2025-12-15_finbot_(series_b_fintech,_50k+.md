# Add Pattern

Compliance-aware Memory in Fintech: When implementing Memory in financial services, ensure stored context includes audit trail metadata (timestamp, conversation ID, user action confirmation). Memory should exclude sensitive PII fields but retain transaction context. This allows compliance teams to verify conversation history while maintaining data minimization principles.
